조건문

    - 프로그래밍 코드가 해석되는 순서를 제어하는 '제어문'중 한 종류
    - 작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

    * 조건식 : 결과가 True OR False인 식
      - 논리 연산자(&&, ||, !)
      - 비교 연산자
      - boolean

if()

    
      number = 1;
      if(number == 1){
        true;
      }

입력 받은수가 양수(0보다 큰수) 인지 확인

if() else

    
      number = 1;
      if(number == 0){
        true;
      }
      else{
        false;
      }

1-100 사이의 난수를 발생시켜 홀/짝 판별하기

if() else if() else

    
      number = 1;
      if(number == 0){
        true;
      }
      else if(number == 1){
        true;
      }
      else{
        false;
      }

Early Return

    return : 현재 싱행중인 함수를 즉시 종료하고 호출한 곳으로 돌아감.

    if 조건에 현재 수행하려는 기능의 예외 상황을 조건식으로 작성하여 
    예외 상황 발생시 즉시 함수를 종료하게 만듬 

    장점 : 유지보수성 향상
    단점 : 함수 코드의 복잡도가 늘어난다

switch, case, default, break

    switch : 
    - 식 하나의 결과로 많은 경우의 수를 처리할때 사용하는 조건문
    - 식의 결과와 일치하는 값을 가진 case 가 수행됨
    - 식의 결과와 일치하는 값을 가진 case 가 없을 경우 default 가 수행됨

    
      switch(i){
        case 1: A code; break;
        case 2: B code; break;
        case 3: C code; break;
        default: break;
      }

입력된 수가 3,4,5 가 맞는지 확인
