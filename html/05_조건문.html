조건문

    - 프로그래밍 코드가 해석되는 순서를 제어하는 '제어문'중 한 종류
    - 작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

    * 조건식 : 결과가 True OR False인 식
      - 논리 연산자(&&, ||, !)
      - 비교 연산자
      - boolean

if()

    
      number = 1;
      if(number == 1){
        true;
      }

입력 받은수가 양수(0보다 큰수) 인지 확인

if() else

    
      number = 1;
      if(number == 0){
        true;
      }
      else{
        false;
      }

1-100 사이의 난수를 발생시켜 홀/짝 판별하기

if() else if() else

    
      number = 1;
      if(number == 0){
        true;
      }
      else if(number == 1){
        true;
      }
      else{
        false;
      }
