up & down game

    1-200 사이 난수를 몇회 만에 맞추는지 카운트 하는 겜

    1) prompt 사용
      - 미입력시 : alert('다시 입력해주세요'); - 카운트 증가 X
      - 잘못 입력한 경우(1-200 사이 X) : alert('1-200 사이 입력해주세요'); - 카운트 증가 X
      - 취소한 경우 : alert(`${count}번에서 포기 하셨습니다 (정답 : ${answer})`); - break;

    2) 기회는 7회 -> 7회 초과시 실패처리 : alert(`7회로 실패 (정답 : ${answer})`); break;

    3) 입력 받은 수가 난수(answer) 보다 높거나 낮은 경우
      - 높은 경우 : alert(`[ DOWN ] 현재 ${count}회`);
      - 낮은 경우 : alert(`[ UP ] 현재 ${count}회`);
      - count++;

    4) 정답인 경우 : alert(`[ 정답입니다 ] 현재 ${count}회`); break;
