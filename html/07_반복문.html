for

    - 프로그램의 흐름을 제어하는 제어문 중 하나
    - 특정 코드를 반복 수행 할 때 사용하는 구문
    - 끝이 정해져 있는 반복에서 주로 사용

    [작성법]
    for(초기식(i=0); 조건식(i>10); 증감식(i++)){
      조건식이 true 일 경우 수행 코드;
    }

이중 for

    반복문 내에 반복문이 작성된 형태

    [작성법]
    for(초기식(i=0); 조건식(i>10); 증감식(i++)){
      for(초기식(j=0); 조건식(j>10); 증감식(j++)){
        code;
      }
    }
