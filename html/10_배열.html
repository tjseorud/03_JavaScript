배열(Array)

    * 변수 : 메모리상에 값을 저장하는 공간(단일)
    * 배열 : 변수를 묶음으로 다루는 것
      - 배열에 저장된 요소를 구분하기 위해서 index(색인) 이용(0부터 시작)
      - JS 배열의 특징
        1. 자료형의 제한 X (여러 자료형 저장 가능, 보통 단일로 통일)
        2. 길이 제한 X (자동으로 배열의 길이가 늘었다, 줄었다 함)
        3. 배열을 위한 다양한 필드(변수)/메서드(함수)가 제공
          ex) Arr.length -> 배열 길이 반환
              Arr.indexOF("값") ->"값"이 몇번 요소에 있는지 (없으면 -1)
  
    [JS 배열 선언 방법]
    1. const arr = [];  
      * [] : 배열기호
    2. const arr = new Array(); 
      * new 연산자 : 새로운 객체 생성
    3. const arr = new Array(); 
      -> 처음 생성될때 5칸 모두 비어있음(empty)
    4. const arr = [값1, 값2, 값3];

Array for

    - 배열에 저장된 요소들은 index를 이용해서 구분
    - index는 0부터 배열 길이 -1 까지 1씩 증가(index는 0 이상 정수)
    - index 값을 증가시켜 배열 내 요소를 접근할 때 효율적인 방법이 for을 이용하는 것

2차원 Array

    * 변수 : 메모리상에 값을 저장하는 공간(단일)
      ex) const i = 1;
    * 1차원 배열 : 변수를 묶음으로 다루는 것(변수 n개)
      ex) const arr = [i];
    * 2차원 배열 : 1차원 배열을 묶음으로 다루는 것(1차원 배열 n개)
      ex) const arr = [[i]];
    * 3차원 배열 : 2차원 배열을 묶음으로 다루는 것(2차원 배열 n개)
