DOM(Document Object Model)

    HTML 문서를 객체 기반으로 표현한 것
    -> html에 작성된 내용을 트리구조로 나타냄.
    -> 각각의 태그, TEXT, 등을 통틀어 Node라고 함.

Node

    1) 부모 노드 탐색 : parentNode
    2) 첫 번째 자식 노드 탐색 : fiestchild
    3) 마지막 자식 노드 탐색 : lastChild

    4) 모든 자식 노드 탐색 : childNodes(배열 형태로 반환)
      -> 원하는 위치에 존재하는 자식노드 탐색 : childNodes[index]
      
    
    5) 이전 형제 노드 탐색 : previousSibling
    6) 다음 형제 노드 탐색 : nextSibling

Element(요소) 탐색 방법

    Node : 요소, 주석, 내용(TEXT)
    Element : 요소(Node의 하위 개념)

    [Element만 탐색하는 방법]
    children : 자식 요소만 모두 선택(배열)
    parentElement : 부모 요소 선택

    firstElementChild : 첫 번째 자식 요소 선택
    lastElementChild : 마지막 자식 요소 선택

    previousElementSibling : 이전 형제 요소 선택
    nextElementSibling : 다음 형제 요소 선택
