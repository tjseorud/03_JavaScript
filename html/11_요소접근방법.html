DOM(Document Object Model)

    - html 문서를 객체 기반 형식으로 표현한 것(트리 구조)

DOM을 이용한 요소 접근 방법

    1. id 속성 값을 이용한 접근
      - document.getElementById("id");
    * HTML의 id(식별자) 속성 : 
      -> 중복 X 
      -> 문서 전체에 하나만 존재(유일한)
      -> 단수(Element) 형태로 작성

    2. class 속성 값을 이용한 접근
      - document.getElementsByClassName("class");
        -> (유사)배열 형태로 반환

    3. tag 속성 값을 이용한 접근
      - document.getElementsByTagName("tag");
        -> (유사)배열 형태로 반환

    4. name 속성 값을 이용한 접근
      - document.getElementsByName("name");
        -> (유사)배열 형태로 반환

    5. CSS 선택자를 이용한 접근
      1) document.querySelector("CSS");
        -> 선택자가 일치하는 요소 중 [첫 번째]만 반환
      2) document.querySelectorAll("CSS");
        -> 선택자가 일치하는 요소 [모두]를 (유사)배열 형태로 반환
